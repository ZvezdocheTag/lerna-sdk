# popup

## Install

```bash
npm install --save popup
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'popup'
import 'popup/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
